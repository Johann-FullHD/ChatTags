# Changelog

## 1.1.0
- Chat format changed to [TAG]<PLAYER> Message
- Tag shown in Tab list and above name via scoreboard teams
- Team color matches tag color (1.13+)
- Added cooldown for changing/clearing tags (configurable)
- New commands: /tag list, /tag clear-all, /tag disable-all
- Admin commands: /tag set <player> <text>, /tag clear <player>
- Join/Quit handlers to update and clean up scoreboard entries
- Internal refactors and minor fixes

## 1.0.0
- Initial release with custom tags, color, toggle, preview, clear
